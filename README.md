# ✍️ Air Writing Using Python

A computer vision-based project that allows users to write or draw in the air using hand gestures. The system tracks hand movements in real time and converts them into digital strokes, creating a touch-free writing experience.

## 🚀 Features

* Real-time hand gesture tracking
* Air writing using finger movement
* Touch-free digital drawing interface
* Live visualization of written strokes
* Simple and interactive user experience

## 🛠️ Technologies Used

* **Python**
* **OpenCV** – Image processing and computer vision
* **NumPy** – Numerical operations and array handling
* **MediaPipe** (if used) – Hand tracking and landmark detection

## 📌 How It Works

1. The webcam captures real-time video input.
2. The system detects and tracks hand movements.
3. Finger position is identified to recognize writing gestures.
4. The movement coordinates are converted into digital strokes.
5. The output is displayed on the screen as air-written text or drawings.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/air-writing-python.git
```

Install required libraries:

```bash
pip install opencv-python numpy mediapipe
```

## ▶️ Run the Project

```bash
python air_writing.py
```

Make sure your webcam is connected and allow camera access.
